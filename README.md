# Employee Data Analysis with SQL in Python

## Project Overview
This project analyzes a company's employee data to provide insights for the HR department. Using **SQL queries in Python**, we explored employee information such as department, salary, and hire date.

## Objectives
- Identify employees in specific departments like Marketing, Sales, and Finance.  
- Find high-earning employees and calculate their monthly salaries.  
- Spot the top performers, most recent hires, and longest-serving employees.  
- Generate clear, department-specific reports for HR decisions.

## Technologies Used
- **Python**  
- **SQLite3** (for database management)  
- **Pandas** (for displaying query results)  
- **Faker** (for generating realistic employee data)

## Key Steps
1. **Create Database & Table** – Set up `employees` table with columns: `employee_id`, `emp_name`, `department`, `salary`, and `hire_date`.  
2. **Generate Sample Data** – Added 100 realistic employee records using the Faker library.  
3. **Task 1: Basic Queries** – Selected employees by department, salary, or multiple departments.  
4. **Task 2: Sorting & Limiting** – Found top earners, recent hires, and oldest employees.  
5. **Task 3: Calculated Fields & Aliases** – Calculated monthly salaries and prepared department-wise reports.

## Conclusion
This project demonstrates how SQL can turn raw employee data into actionable insights. HR can now easily see salary patterns, track hiring trends, and make informed decisions for promotions, payroll planning, and workforce management.
